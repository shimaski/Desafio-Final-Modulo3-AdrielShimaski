# Análise SWOT - Segurança TechCorp Solutions

## **Forças**
- Sistema de backups em funcionamento  
- SSL configurado no ambiente  
- Serviços essenciais respondendo adequadamente  

## **Fraquezas**
- FTP com acesso anônimo ativo  
- Vulnerabilidade crítica de SQL Injection  
- Credenciais hardcoded em arquivos de configuração  
- Histórico do bash exposto com comandos sensíveis  
- Arquivos de configuração acessíveis via web  

## **Oportunidades**
- Implementação de WAF e controles de segurança  
- Programa de conscientização em segurança  
- Automação de testes de segurança  
- Adoção de framework de segurança  

## **Ameaças**
- Concorrentes podem explorar vulnerabilidades  
- Vazamento de dados sensíveis (credenciais, API keys)  
- Possibilidade de ataques ransomware  
- Multas por não conformidade com regulamentações
